## 📌 Project Overview

This project analyzes Myntra fashion product data to identify patterns in:

- Popular Brands
- Product Pricing
- Customer Ratings
- Premium Fashion Brands

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

| Metric | Value |
